Metric convention

\[g_{\mu \nu }=\text{diag} (-1,1,1,1) \]

Geometry of the universe.

FRW metric

\begin{align*} ds^2 &= -c^2 dt^2 + a^2 (t)(dx^2 +dy^2 +dz^2 ) \\ &= -c^2 dt^2 +a^2 (t) \left[ \frac{dr^2 }{1-k\frac{r^2 }{R_0 ^2 }}+r^2 d\Omega ^2 \right] \end{align*}

Metric on $S^2 $:

\[d\Omega ^2 \equiv d\theta ^2 +\sin ^2 \theta d\phi ^2 \]

Hubble parameter

\[H\equiv \frac{\dot{a}}{a} \]

Conformal time

\[d\eta =\frac{dt}{a(t)} \]

Geodesic equation:

\[\frac{d^2 x^{\mu }}{d\tau ^2 }=-\Gamma ^{\mu }_{\alpha \beta }\frac{dx^{\alpha }}{d\tau }\frac{dx^{\beta }}{d\tau } \]

Christoffel symbols:

\[\Gamma ^{\mu }_{\alpha \beta }=\frac{1}{2}g^{\mu \lambda } \left( \partial _{\alpha }g_{\beta \lambda }+\partial _{\beta }g_{\alpha \lambda }-\partial _{\lambda }g_{\alpha \beta } \right) \]

Covariant derivative:

\[\nabla _{\alpha }V^{\mu }=\partial _{\alpha }V^{\mu }+\Gamma ^{\mu }_{\alpha \beta }P^{\beta } \]

Redshift:

\[z=\frac{\lambda _0 -\lambda _1 }{\lambda _1 } \]

Redshift-scale factor relation:

\[1+z=\frac{1}{a} \]

Comoving distances:

\[\chi (z)=c\int_{t_1 }^{t_0 } \frac{dt}{a} \]
